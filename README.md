此平衡小车使用主控单片机STM32F103C8T6进行控制，电机选取gm25-310带编码器进行测速，挂载MPU6050进行姿态数据读取，电源使用7.4V电源给电机供电，再接5V降压模块给单片机供电，小车还挂载了HC06蓝牙模块方便手机操控,此外接了OLED以及保留了串口位置方便调试
